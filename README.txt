how to start digidoc4j development

git clone --recursive https://github.com/open-eid/digidoc4j.git

cd digidoc4j
ant

User stories are described here: https://www.pivotaltracker.com/s/projects/1110130

To load libraries (including new and updated libraries) without performing a complete build: ant load.dependencies

javadoc you can find: http://open-eid.github.io/digidoc4j/
